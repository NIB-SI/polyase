Introduction
------------

**polyase** is a Python package for analyzing allele-specific expression (ASE) from long-read RNA-seq data in polyploid organisms. It provides tools for loading ASE data, filtering, statistical analysis, and visualization of ASE patterns across different conditions or tissues.



Installation
-------------
The package can be installed via pip:

.. code-block:: bash

      pip install polyase
