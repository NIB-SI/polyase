# Polyase

A Python package for analyzing allele-specific expression in polyploid organisms.

## Installation

You can install polyase using pip:
pip install -i https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple/ polyase==0.1.7




On Mac you need to install pyranges before installing the package.
